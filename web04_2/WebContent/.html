표현식(Expression)

1. jsp의 표현식 문자 : <%="Hello" %>

2. jsp의 표현식 숫자 : <%=20 %>

3. jsp의 표현식 문자열변수: <%=name %>

4. jsp의 표현식 숫자변수: <%=num %>

5. jsp의 표현식 연산문 : <%=name+num %>

6. jsp의 표현식 비교 : <%=num1>num2 %>

7. jsp의 표현식 연산문 : <%=num1+40 %>

8. jsp의 표현식 연산문 : <%=num1%3 %>

9. jsp의 표현식 연산문 : <%=num1>40 ? "크다" : "작다" %>

<%--

10. jsp의 표현식 객체 접근 : <%=vo.getCusid() %>

--%>

표현식은 각종 함수나 클래스 적용이 되지 않으며, 객체의 활용이 자유롭지 못함

표현 언어(Expression Language)

1. jsp의 표현언어 문자 : ${name }

2. jsp의 표현언어 문자 : ${num }

1. jsp의 표현언어 연산문 : ${num1+num2 }

1. jsp의 표현언어 연산문 : ${name+num1 }
